# Project Design Philosophies:
* **Steal aesthetics and colors from the main VRChat website. Duh.**
* **W i d e r, less deadspace** (FUCK deadspace, all my homies hate deadspace)
* **Give users plenty of options to configure their ideal look** (and section off/comment everything well enough to make it fairly easy to understand what's going on, so end users who REALLY hate my design decisions can just yeet 'em)
* **Enhance readability of text** (larger font size & better margins/padding) **and profile pictures** (larger size, rounded squares > circles, disparity between defined-pic and undefined-pic users)
* **Make sites feel more** ***responsive*** **through Animation tuning and special effects**
